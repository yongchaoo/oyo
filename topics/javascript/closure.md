# 什么是闭包？有哪些应用场景？

### 标准答案

1. **定义**：函数能访问其词法作用域中的变量，即使函数在作用域外执行
2. **本质**：函数 + 它引用的外部变量 = 闭包
3. **形成条件**：内部函数引用了外部函数的变量，且内部函数被返回或传递到外部

### 应用场景

- 数据封装/私有变量：模块模式，IIFE
- 函数工厂：`makeAdder(5)` 返回一个加 5 的函数
- 回调和事件处理：setTimeout、addEventListener 中保持对外部变量的引用
- 柯里化和偏函数

### 追问点

- 闭包与内存泄漏：闭包引用的变量不会被 GC，滥用导致内存无法释放
- 经典循环问题：`for(var i=0;i<5;i++) setTimeout(...)` 全输出 5，用 let 或 IIFE 解决
